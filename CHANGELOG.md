# Changelog

## [v2.1] - 2026-05-07

### 错误报告优化
- Summary CSV 的 error 列现在显示可读的中文原因，而非原始异常堆栈
- 区分四类失败原因：网络超时 / 化学品未找到 / 滤盒未找到 / 下拉选项不存在
- 温度、工作强度找不到匹配时，报错并列出网站实际可用选项，方便修正 CSV

### 多化学品适配
- Cookie 横幅：若在化学品搜索过程中再次弹出，自动关闭
- CAS 验证：点击 checkbox 前先确认搜索结果包含正确 CAS 号，防止误选
- 单位下拉框：`get_by_label("单位")` 失败时自动改用选项内容匹配，适配甲醛等特殊化学品
- 暴露浓度输入框：改为正则匹配"暴露"，兼容有/无突破浓度字段两种页面结构
- 单位识别：改为文本模糊匹配，支持 `ppm`、`mg/m3`、`mg/m³` 等写法

### Bug 修复
- 修复各步骤错误标签（op）未及时更新，导致报错信息指向错误步骤的问题

---

## [v2.0] - 2026-05-06

### 首次发布
- Playwright 自动化批量查询 3M SLS 使用寿命
- 支持 CSV 批量输入（CAS、暴露浓度、单位、滤盒型号、环境参数）
- 自动下载 PDF，生成 summary.csv 汇总结果
- 使用系统已安装的 Chrome / Edge，无需额外安装浏览器
- 静默后台模式 / visual 调试模式可选
- 失败任务自动重试一次
- 使用 `load` 策略替代 `networkidle`，适配网络波动环境
